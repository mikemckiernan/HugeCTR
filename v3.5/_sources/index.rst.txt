Merlin HugeCTR
==============

Merlin HugeCTR is an open source library and provides a GPU-accelerated recommender framework
that is designed to distribute training across multiple GPUs and nodes and estimate click-through rates (CTRs).

For more information, see the `Introduction <hugectr_user_guide.html>`_.
